# osu! Mania PP 계산기

## 소개
osu! Mania에서 등수를 산정할때 필요한 점수인 PP(Perfomance Point)는 일반적으로 유저에게 계산식을 제공해주지 않습니다.
그래서 점수 계산식을 찾아 유저가 자신이 획득할 PP를  계산할 수 있도록 앱으로 제작했습니다.
